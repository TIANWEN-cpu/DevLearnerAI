# 第一章：Python 基础语法全解

## 1. 变量与命名规范
变量名必须以字母或下划线开头，不能以数字开头。推荐使用 `snake_case` 命名法。

```python
user_name = "Claude"
user_age = 1
```

## 2. 核心数据类型
- **Int**: 整数，如 `10`, `-5`
- **Float**: 浮点数，如 `3.14`
- **String**: 字符串，如 `"Hello"`
- **Bool**: 布尔值，`True` 或 `False`

## 3. 运算符
- 算术：`+`, `-`, `*`, `/`, `//` (整除), `%` (取余), `**` (幂)
- 逻辑：`and`, `or`, `not`
