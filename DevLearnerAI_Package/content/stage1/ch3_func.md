# 阶段一：基础语法与思维 - 函数与模块

## 1. 函数定义与调用
- `def` 关键字。
- 参数类型：位置参数、关键字参数、默认参数、动态参数 (`*args`, `**kwargs`)。
- 返回值：`return` (支持返回多个值)。

## 2. 变量作用域
- `L` (Local) -> `E` (Enclosing) -> `G` (Global) -> `B` (Built-in)。
- `global` 与 `nonlocal` 关键字的使用场景。

## 3. 匿名函数与高阶函数
- `lambda` 表达式。
- `map()`, `filter()`, `reduce()`。
- 装饰器 (Decorator) 基础：函数闭包的应用。

## 4. 模块与包
- `import` 机制与 `sys.path`。
- `__init__.py` 的作用。
