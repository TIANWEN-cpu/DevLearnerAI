# 阶段一：基础语法与思维 - 逻辑控制

## 1. 条件判断
- `if`, `elif`, `else` 结构。
- 逻辑运算符：`and`, `or`, `not`。
- 成员运算符：`in`, `not in`。

## 2. 循环结构
- `for` 循环与 `range()` 函数。
- `while` 循环与死循环规避。
- 循环控制：`break`, `continue`, `pass`。
- `enumerate()`：同时获取索引与值。

## 3. 列表推导式 (List Comprehension)
- 语法：`[expression for item in iterable if condition]`。
- 简洁且高效的集合创建方式。
