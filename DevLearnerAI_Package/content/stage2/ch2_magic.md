# 阶段二：面向对象编程 (OOP)

## 1. 类与对象
- `class` 定义与 `__init__` 构造方法。
- `self` 关键字的本质与作用。
- 属性访问：私有属性 (`__attr`) 与 `property` 装饰器。

## 2. 继承与多态
- 单继承、多继承与 `super()` 调用。
- 方法重写 (Override)。
- `isinstance()` 与 `issubclass()` 类型检查。

## 3. 魔术方法 (Magic Methods)
- `__str__` / `__repr__`: 自定义对象字符串展示。
- `__call__`: 使对象可调用。
- `__getitem__` / `__setitem__`: 容器化操作。
- `__enter__` / `__exit__`: 上下文管理器 (`with` 语句)。

## 4. 类的高级特性
- `@staticmethod` 与 `@classmethod` 的区别。
- 抽象基类 (ABC): `abc` 模块的应用。
