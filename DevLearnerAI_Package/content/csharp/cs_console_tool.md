# 用 C# 做一个控制台小应用

## 学习目标
- 把类、集合、输入输出和文件处理真正串起来
- 做出一个最小但完整的控制台应用
- 体验“从课程题走向作品”的那一步

## 一个合适的方向
- 学习记录器
- 简单记账本
- 命令行联系人管理器

这些项目的共同特点是：  
小而完整，适合练结构。

## 做的时候别着急做大
先把这三件事做好：
- 数据结构清楚
- 流程入口清楚
- 结果能保存下来
