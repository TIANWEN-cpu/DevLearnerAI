# DevLearner AI 使用教程

## 1. 快速上手
- **学习中心**：在左侧列表选择章节，系统将自动加载 Markdown 内容并记录你的进度。
- **代码实操**：在编辑器中输入 Python 代码，点击“运行代码”看输出结果。完成任务后点击“自动评分”。
- **数据库管家**：随时刷新查看 SQLite 数据库的变化，观察 `students` 表的状态。

## 2. 玩转 AI 助手 (AI Mentor)
- **配置 API**：在右侧侧边栏输入你的 API Key。支持 Claude/OpenAI 等主流接口。
- **智能纠错**：如果你在练习中报错，直接点击 AI 助手询问。 
- **代码解析 (Ctrl+Shift+A)**：在编辑器选中一段代码并按下快捷键，AI 会瞬间解释这段逻辑。

## 3. 常见问题
- **评分标准**：评分基于代码逻辑、PEP8 规范和执行效率。 
- **数据保存**：系统自动将学习章节完成情况存入本地 `db/dev.db`。
