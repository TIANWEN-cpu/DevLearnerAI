# 项目十五：Python 命令行工具打包与分发起步

## 项目目标

把一个已经能跑的 Python 小工具，整理成：

- 可通过命令行调用
- 参数清晰
- 可被他人安装和运行

## 为什么这个项目值得做

很多人会写脚本，但不会把脚本变成真正可交付的小产品。  
这个项目就是帮你跨过这一步。

## 推荐 MVP 范围

- 选一个已有脚本
- 补齐 `argparse`
- 设计 `--help`
- 规范项目结构

## 可继续扩展

- 增加入口脚本
- 增加 README 使用说明
- 打包成可分发形式

## 验收标准

- 运行方式清晰
- 参数说明完整
- 他人拿到后能按说明运行
