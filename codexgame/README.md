# Echo Lock

一个零依赖、直接打开即可运行的 Canvas 原型游戏。

## 启动方式

直接打开 `index.html` 即可。

如果浏览器因为本地文件策略阻止脚本，也可以在当前目录运行：

```powershell
python -m http.server 8000
```

然后访问 `http://localhost:8000/`。

## 操作

- `WASD`: 移动
- `鼠标`: 瞄准
- `左键` 或 `Space`: 射击
- `R`: 放出一个回响，重演最近 4 秒
- `Enter`: 重开当前关 / 通关后重开整局

## 核心规则

- 回响会重放你最近 4 秒的行动。
- 回放结束后，它会在终点额外停留 3 秒。
- 回响最多同时存在 2 个。
- 踏板可以被玩家或回响激活。
