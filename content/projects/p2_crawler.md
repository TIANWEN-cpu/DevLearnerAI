# 实战项目二：多线程图片爬虫

## 任务描述
编写一个能够从网页抓取图片并保存到本地的爬虫，要求支持多线程下载以提高效率。

## 技术要点
- **requests**: 发送 HTTP 请求，获取网页源码。
- **BeautifulSoup**: 解析 HTML，提取图片 URL。
- **threading**: 实现多线程并发下载。
- **os**: 处理本地文件路径与存储。

## 核心逻辑
1. 获取目标网页 HTML。
2. 匹配所有的 `<img>` 标签及 `src` 属性。
3. 创建线程池或手动启动多个线程进行下载任务。
4. 加入异常处理（Timeout, 404 等）。
