# 阶段一：Python 简介与环境搭建

## 1. 为什么选择 Python？
Python 是一种**解释型、面向对象、动态数据类型**的高级程序设计语言。其语法优美、代码简洁（优雅胜过丑陋，简单胜过复杂）。

## 2. 环境安装
- **Python 解释器**: 建议安装 3.10+ 版本。
- **包管理**: 使用 `pip` 安装第三方库。
- **IDE**: 推荐 VS Code 或 PyCharm，当然你现在正在使用的 **DevLearner AI** 也是绝佳的练习场。

## 3. 第一个程序
```python
print("Hello, DevLearner!")
```
在右侧「实战实验室」输入上述代码并点击「运行」，即可开启你的编程之旅。
