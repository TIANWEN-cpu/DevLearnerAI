# 项目七：SQL 报表与学习数据看板

## 项目目标
把练习记录、完成情况和学习时长做成可解释的报表结果。

## 这个项目为什么重要
- 指标意识
- SQL 聚合与窗口函数
- 结果解释能力
- 数据展示意识

## 推荐 MVP 范围
- 完成数指标
- 平均分指标
- 最近 7 天活跃情况
- 主线完成度排行

## 拆分建议
- 先定义要回答的业务问题
- 再写查询，不要先写 SQL 再找意义
- 最后把结果整理成固定结构

## 验收标准
- 至少输出 4 个核心指标
- 每个指标都能解释来源
- 查询结果能稳定复现
