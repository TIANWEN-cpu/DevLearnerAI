# 变量、类型与表达式

## 学习目标
- 理解变量不是“盒子”，而是名字绑定到某个值
- 区分常见基础类型
- 写出可靠的表达式和输出格式

## 变量是什么
在 Python 里，变量更像“标签”。例如：

```python
age = 18
```

这句话不是“往盒子里装 18”，而是让名字 `age` 指向整数值 `18`。

重新赋值时：

```python
age = age + 1
```

旧值不会被“改写成表达式”，而是先计算右边，再把新结果绑定回 `age`。

## 常见基础类型
- `int`：整数
- `float`：浮点数
- `str`：字符串
- `bool`：布尔值

```python
name = "Alice"
score = 98
ratio = 0.85
passed = True
```

## 表达式如何求值
表达式就是“会被算出结果”的代码片段：

```python
total = 3 + 5 * 2
```

这里会先算乘法，再算加法，结果是 `13`。

## 字符串格式化
推荐优先使用 f-string：

```python
name = "Alice"
score = 95
print(f"{name} 的成绩是 {score}")
```

它比拼接字符串更清晰，也更不容易出错。

## 类型转换
输入和外部数据经常需要转换：

```python
text = "42"
value = int(text)
```

转换失败时会报错，所以后面学习异常处理非常重要。

## 常见错误
- 把数字和字符串直接相加
- 不知道 `input()` 得到的是字符串
- 把 `=` 和 `==` 混淆

## 小练习
```python
name = "Bob"
age = 20
print(f"{name} 明年 {age + 1} 岁")
```

你可以继续尝试：
- 把 `age` 改成字符串看看会发生什么
- 用 `type()` 打印变量类型

## 本课总结
变量、类型和表达式是编程最底层的语法肌肉。后面流程控制、函数、数据库字段设计，本质上都建立在“数据长什么样”之上。
