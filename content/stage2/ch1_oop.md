# 阶段二：面向对象与高级特性

## 1. 类与对象
- `__init__`、实例属性与类属性。
- 实例方法/类方法 (@classmethod)/静态方法 (@staticmethod)。
- 继承、`super()` 的使用、MRO (方法解析顺序)。
- 封装与私有属性（_ 与 __ 的区别）。

## 2. 高级特性
- 迭代器与生成器 (`yield`)。
- 装饰器：闭包 + @ 语法。
- 上下文管理器：`__enter__`/`__exit__`。
- 元类：`type` 与 `__new__`。

## 3. Python 数据模型
- 魔术方法：`__str__`, `__repr__`, `__call__`, `__getitem__`。
