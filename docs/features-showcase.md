# DevLearner AI -- 功能全景展示

本文档详细介绍 DevLearner AI 的每一项核心功能，帮助你全面了解这款编程学习平台能做什么。

## AI 智能导师

**不是通用聊天机器人，是你的专属编程辅导老师。**

### 上下文感知

AI 导师能够感知你的学习状态：

- **当前课程**：知道你正在学哪门课、哪个知识点
- **练习进度**：知道你做到了哪道题、之前的提交结果
- **历史对话**：加载最近 12 条消息作为上下文，保持对话连贯
- **知识库**：基础知识库 + 个性知识库 + 扩展文件知识库，三层知识体系

### 双模式工作

| 模式 | 场景 | 说明 |
|:---|:---|:---|
| AI 工作台 | 深度学习 | 独立页面，多会话管理，按主题拆分对话 |
| 侧边助手 | 即时答疑 | Dock 面板，不离开当前课程页面 |

### 快捷提问

一键触发常用场景：

- **解释当前课程** -- 选中一段看不懂的内容，让 AI 逐行讲解
- **分析当前代码** -- 把练习区的代码发给 AI，获得改进建议
- **拆解当前项目** -- 把项目需求发给 AI，获得分步实现方案

### 技术细节

- 支持 OpenAI 兼容 API，可自定义端点和模型
- Markdown 渲染 + 代码语法高亮
- 多会话持久化存储（SQLite）
- API 密钥通过 Windows Credential Manager 安全存储

## 代码执行沙箱

**在安全的环境中运行代码，5 层防护让你放心练。**

### Python 实际运行

不是只做静态检查，而是真正执行代码：

1. **AST 预检** -- 解析代码的抽象语法树，拦截 `os.system`、`subprocess`、`eval`、`exec` 等危险调用
2. **内置函数限制** -- 白名单机制，只允许 `math`、`json`、`re` 等安全标准库
3. **属性访问控制** -- 禁止访问 `__class__`、`__bases__`、`__subclasses__` 等双下划线属性
4. **文件操作隔离** -- 限制在临时工作目录内，不能访问系统文件
5. **资源限制** -- 标准输出 12KB 上限，执行 3 秒超时

### 多语言评测

| 语言 | 评测方式 |
|:---|:---|
| Python | 沙箱实际运行 + 测试用例比对 |
| SQL | 内存数据库真实执行 + 结果比对 |
| C / C++ | 关键字结构检查 + 代码规范验证 |
| C# | 关键字结构检查 + 代码规范验证 |

### 安全边界测试

项目包含专门的沙箱逃逸安全测试（`test_security_sandbox_escape.py`），覆盖：

- `eval()` / `exec()` 拦截
- `__builtins__` 访问拦截
- `__import__` 绕过尝试
- 文件系统逃逸尝试
- 资源耗尽攻击防护

## 沉浸式课程体系

**130 个课时，从零基础到工程实战的完整知识地图。**

### 8 条学习路线

| 路线 | 课时数 | 覆盖内容 |
|:---|:---:|:---|
| **Python** | 20+ | 语法基础 / 数据类型 / 控制流 / 函数 / OOP / 模块 / 文件操作 / 异步 / 测试 / 调试 |
| **C** | 12+ | 数据类型 / 指针 / 内存管理 / 字符串 / 结构体 / 链表 / 文件 / Make/GDB |
| **C++** | 6+ | 语法与函数 / 字符串与 vector / 迭代器与循环 / OOP 与类 / RAII 与智能指针 / 文件 IO |
| **C#** | 10+ | 基础语法 / 方法与集合 / OOP / LINQ / 委托与事件 / 异步与文件 / Web API / 调试 |
| **数据库** | 12+ | 概念 / CRUD / 聚合 / 子查询 / 索引与事务 / 窗口函数 / Schema 设计 / 查询优化 / Python 集成 |
| **算法** | 10+ | 复杂度基础 / 数组与哈希 / 二分与贪心 / 栈与队列 / 动态规划 / 图最短路 / 前缀和与滑窗 |
| **融合项目** | 8+ | C 项目 / C# 项目 / SQL 报表 / FastAPI 数据 / 日志分析 / API 客户端 / 算法 Playground |
| **实战项目** | 10 | Todo CLI / 数据分析 / Web 应用 / AI 对话助理等递进式项目 |

### 课程结构

每个课时以 Markdown 格式组织：

```
Track（技术栈）
  └── Module（模块）
        └── Lesson（课时）
              ├── 学习目标
              ├── 前置条件
              ├── 正文内容（Markdown + 代码块）
              ├── 代码示例
              └── 关联练习
```

### 元数据管理

- `content/metadata/course_map.json` -- 课程元数据统一管理
- 每课附带难度、预估学时、前置依赖和学习目标
- 支持课程完成度追踪和笔记功能
- 内容服务延迟加载，减少启动耗时

## 交互式练习

**176 道精选练习题，覆盖全部 8 条技术路线。**

### 练习体系

| 维度 | 说明 |
|:---|:---|
| **数量** | 176 道，持续扩充 |
| **语言** | Python / C / C# / SQL / C++ |
| **难度** | 基础 / 进级 / 综合三级标签 |
| **覆盖** | 全部 8 条学习路线 |

### 评测维度

每道练习的评测不是简单的对错判断，而是多维度评分：

1. **语法检查** -- 代码是否符合语言语法
2. **结构检查** -- 是否使用了要求的编程结构（如循环、函数、类）
3. **运行结果** -- Python 代码实际运行，SQL 真实查询验证
4. **测试用例通过率** -- 多组测试数据的通过比例

### 用户体验

- **练习中心**：按技术栈和难度筛选，快速找到目标练习
- **草稿自动保存**：写到一半关闭窗口，下次打开自动恢复
- **练习历史**：每次提交的代码和评测结果永久保存
- **代码编辑器**：语法高亮 + 自动缩进，桌面原生体验

## 工程化特性

DevLearner AI 不只是功能丰富，在工程质量上也达到了专业水准。

### 代码质量

- **Ruff** 统一 lint + format（替代 flake8 + isort + black）
- **pytest** 测试框架，1000+ 测试用例
- 测试覆盖沙箱安全、数据库、评测逻辑、AI 模块、集成流程和安全边界

### CI/CD

- GitHub Actions 支持 Python 3.9 + 3.12 矩阵测试
- 覆盖率报告，最低阈值 40%
- 自动化 lint + format check + pytest

### 构建系统

- 统一构建脚本 `scripts/build/build.py`
- 支持 release / dev / codex 三种打包变体
- 自动 `.spec` 文件生成
- dry-run 模式验证构建配置

### 版本管理

- 单一来源：`pyproject.toml` 的 `version` 字段
- `app/config.py` 通过 `importlib.metadata` 动态读取
- 消除多处版本号硬编码

## 安全设计

| 安全层 | 措施 |
|:---|:---|
| 代码执行 | AST 预检 + 内置函数白名单 + 属性访问控制 |
| 文件系统 | 操作限制在临时目录 |
| API 密钥 | Windows Credential Manager 存储 |
| 网络通信 | HTTPS 强制 + TLS 证书验证 |
| 数据库 | WAL 模式 + 写操作锁 + 线程安全 |
| HTML 输出 | 白名单净化，防止 XSS 注入 |
| 资源限制 | 输出 12KB / 执行 3 秒 / 评测 4 秒 |
