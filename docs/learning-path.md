# 推荐学习路径

本文档为 DevLearner AI 的课程内容提供推荐学习顺序，帮助学习者高效规划学习计划。

## 路径总览

课程内容分为 8 条学习路线，建议按照以下优先级和顺序学习：

```
Python 主线 ─────────────────────────────────┐
C 语言路线 ───────────────┐                   │
C++ 路线   ─────────┐     │                   │
                     ▼     ▼                   ▼
算法/数据结构路线 ─────────────► 融合项目路线 ──► 实战项目集
                                              ▲
数据库路线 ────────────────────────────────────┘
                                              ▲
C# 路线 ──────────────────────────────────────┘
```

---

## 路径一：Python 主线（推荐首选）

适合零基础或初级学习者，是项目实战能力的核心路线。

| 阶段 | 模块 | 预计学时 | 核心能力 |
|------|------|---------|---------|
| 1 | 基础模块 · 语法与表达 | 3.5h | 变量、流程控制、函数、文件 I/O |
| 2 | 核心模块 · 数据容器与迭代 | 4h | 列表、字典、集合、迭代器、推导式 |
| 3 | 进阶模块 · 面向对象与类型 | 4h | 类、继承、魔术方法、类型注解 |
| 4 | 高级模块 · 并发编程与性能 | 4.5h | GIL、线程、进程、asyncio、性能优化 |
| 5 | 实战模块 · 框架与项目 | 5h | FastAPI、SQLAlchemy、Web 工程 |

**总计约 21 小时** | 21 课时 | 22 道练习

---

## 路径二：数据库路线

建议与 Python 主线并行学习，数据库是后端开发的必备技能。

| 阶段 | 模块 | 预计学时 | 核心能力 |
|------|------|---------|---------|
| 1 | 基础模块 · SQL 入门 | 3h | CRUD、约束、外键 |
| 2 | 核心模块 · 查询与优化 | 4h | JOIN、子查询、索引、EXPLAIN |
| 3 | 进阶模块 · 高级特性 | 4h | 窗口函数、CTE、视图、事务 |
| 4 | 运维模块 · 生产实践 | 3.5h | 迁移、备份恢复、连接池、性能调优 |

**总计约 14.5 小时** | 16 课时 | 34 道练习

---

## 路径三：C 语言路线

适合想理解底层机制的学习者，建议有 Python 基础后再学习。

| 阶段 | 模块 | 预计学时 | 核心能力 |
|------|------|---------|---------|
| 1 | 基础模块 · 语法与编译 | 4h | 编译流程、控制流、数组、指针 |
| 2 | 核心模块 · 内存与数据结构 | 4.5h | 动态内存、结构体、链表、文件 I/O |
| 3 | 进阶模块 · 系统编程 | 4h | 预处理器、UB、POSIX、调试工具 |
| 4 | 实战模块 · 项目与工程 | 5.5h | Makefile、迷你 libc、内存池、综合项目 |

**总计约 18 小时** | 17 课时 | 32 道练习

---

## 路径四：C++ 路线

建议先完成 C 语言路线，或至少熟悉指针和内存管理。

| 阶段 | 模块 | 预计学时 | 核心能力 |
|------|------|---------|---------|
| 1 | 基础模块 · 语法与核心概念 | 4h | 编译、函数、引用、指针、数组 |
| 2 | 面向对象模块 · 类与对象 | 4.5h | 类、构造/析构、继承、多态、抽象类 |
| 3 | STL 标准库模块 | 4.5h | vector、迭代器、map/set、算法、Lambda |
| 4 | 现代 C++ 模块 | 5h | RAII、智能指针、移动语义、模板、并发 |
| 5 | 进阶与实战模块 | 5h | 文件 IO、异常、构建调试、设计模式、项目 |

**总计约 23 小时** | 25 课时 | 15 道练习

---

## 路径五：C# 路线

适合有面向对象基础的学习者，建议先完成 Python 主线前 3 个阶段。

| 阶段 | 模块 | 预计学时 | 核心能力 |
|------|------|---------|---------|
| 1 | 基础模块 · 语法与 .NET 生态 | 4h | .NET SDK、类型系统、方法、集合 |
| 2 | 核心模块 · 面向对象与数据 | 4h | OOP、泛型、委托、LINQ |
| 3 | 进阶模块 · 异步与文件 | 4h | async/await、文件 I/O、异常调试 |
| 4 | 实战模块 · Web 与工程 | 4.5h | ASP.NET Core、EF Core、DI、测试、部署 |

**总计约 16.5 小时** | 16 课时 | 30 道练习

---

## 路径六：算法/数据结构路线

建议在至少掌握一门语言后开始学习。

| 阶段 | 模块 | 预计学时 | 核心能力 |
|------|------|---------|---------|
| 1 | 基础模块 · 数据结构基础 | 4h | 复杂度、数组、链表、栈、队列、哈希表 |
| 2 | 核心模块 · 树与图 | 4.5h | 二叉树、BST、堆、BFS/DFS、最短路径 |
| 3 | 算法策略模块 | 4h | 排序、二分、动态规划、贪心、回溯 |

**总计约 12.5 小时** | 13 课时 | 17 道练习

---

## 路径七：融合项目路线

建议在完成 Python 主线 + 数据库路线后开始。

| 阶段 | 模块 | 预计学时 | 核心能力 |
|------|------|---------|---------|
| 1 | 综合项目模块 | 6h | 算法可视化、API 客户端、日志分析 |
| 2 | 语言专项模块 | 4h | C 项目、C++ 项目、C# 项目 |
| 3 | 竞赛与拓展模块 | 4h | 蓝桥杯、数学建模、嵌入式、Simulink |

**总计约 14 小时** | 12 课时 | 15 道练习

---

| 中级 | Django 个人博客 | 综合 | 4h |
| 中级 | PyQt5 音乐播放器 | 综合 | 3.5h |
| 高级 | Selenium 自动化测试 | 综合 | 3h |

## 推荐组合路径

### 零基础入门（约 3-4 个月）
```
Python 基础(阶段1-2) → 数据库基础(阶段1) → 融合项目(算法可视化)
→ Python 进阶(阶段3) → 实战项目(Todo工具、数据分析)
```

### 后端工程师（约 4-6 个月）
```
Python 全线 → 数据库全线 → 融合项目全线 → 实战项目(FastAPI、Django)
```

### 系统编程方向（约 4-6 个月）
```
Python 基础 → C 全线 → C++ 全线 → 算法全线 → 实战项目(ORM、内存池)
```

### 全栈工程师（约 6-8 个月）
```
Python 全线 → 数据库全线 → C# 全线 → 算法基础 → 融合项目全线 → 实战项目全线
```
