# 内容扩展总表（第一阶段）

## 这轮先做了什么
- 把课程主线扩展为 `Python / C / C# / 数据库 / 融合项目`
- 把模块组织方式升级为 `基础模块 / 精选模块 / 兴趣模块`
- 给第一批新增课程统一加入 `参考文献 / 官网链接 / 推荐论文`
- 给题库扩了新的语言方向，并为 `C / C#` 增加了第一批结构型练习

## 当前第一阶段目标
- 先把“学习系统的骨架”做扎实
- 让每条主线都有可点进去的真实内容
- 让用户打开程序后，能明显感到内容不是只有几节课和几道题

## 第一阶段新增方向
### Python
- 基础模块：字符串、容器、模块边界、调试工作流
- 精选模块：迭代器与生成器
- 兴趣模块：自动化脚本、HTTP 与 API

### C
- 基础模块：编译、类型、控制流、函数、数组
- 精选模块：指针、内存、结构体、文件
- 兴趣模块：命令行小工具

### C#
- 基础模块：.NET 环境、方法、集合、类与对象
- 精选模块：LINQ、泛型、异常、异步
- 兴趣模块：控制台工具

### 数据库
- 基础模块：子查询、集合操作、索引、事务
- 精选模块：查询计划
- 兴趣模块：报表案例

### 融合项目
- 日志分析器
- C# 学习记录控制台应用
