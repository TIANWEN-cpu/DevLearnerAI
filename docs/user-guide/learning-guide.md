## 学习路线概览

DevLearner AI 提供 **8 条学习路线**，共计 **130+ 课时**，覆盖从基础语法到工程实战的完整知识体系。

| 路线 | 说明 | 难度范围 |
|------|------|----------|
| Python | 语言基础、数据结构、控制流、函数、模块、面向对象、异步编程 | 入门 - 进阶 |
| C 语言 | 基础语法、指针、内存管理、数据结构 | 入门 - 进阶 |
| C++ | 面向对象、STL、模板、现代 C++ 特性 | 入门 - 高级 |
| C# | .NET 基础、面向对象、LINQ、异步编程 | 入门 - 进阶 |
| 数据库 | SQL 基础、数据库设计、SQLite、ORM | 入门 - 进阶 |
| 算法 | 排序、查找、动态规划、图论、数据结构 | 入门 - 高级 |
| 融合项目 | 跨语言综合项目 | 进阶 |
| 实战项目 | 从 Todo CLI 到 AI 对话助理的真实项目 | 进阶 - 高级 |

### 推荐学习顺序

**零基础学员**：
1. Python 入门（基础语法 -> 数据类型 -> 控制流 -> 函数）
2. 数据库基础（SQL 入门 -> 基本查询 -> 数据库设计）
3. Python 进阶（面向对象 -> 模块 -> 文件操作）
4. 融合项目

**有编程基础的学员**：
1. 可直接选择感兴趣的技术栈
2. 建议先浏览课程元数据了解难度和前置要求

## 课程结构说明

课程采用三级层次结构组织：

```
技术栈（Track）
  ├── 模块（Module）
  │     ├── 课时 1（Lesson）
  │     ├── 课时 2（Lesson）
  │     └── 课时 3（Lesson）
  └── 模块（Module）
        └── ...
```

每个课时包含以下元数据：

| 属性 | 说明 |
|------|------|
| 标题 | 课时名称 |
| 难度 | 入门 / 进阶 / 高级 |
| 预估学时 | 建议学习时间 |
| 前置依赖 | 需要先完成的课时 |
| 学习目标 | 完成本课后应掌握的技能 |
| 文件路径 | 对应的 Markdown 课程文件 |

课程元数据统一管理在 `content/metadata/course_map.json` 文件中。

## 开始学习

### 第 1 步：选择学习路线

1. 在左侧导航栏中点击 **学习路径**
2. 浏览可用的技术栈路线
3. 点击你感兴趣的技术栈进入

### 第 2 步：选择模块和课时

1. 在技术栈页面中，浏览模块列表
2. 每个模块显示包含的课时数量和完成状态
3. 点击具体课时开始学习

### 第 3 步：阅读课程内容

课程内容以 Markdown 格式渲染，包含：
- 理论讲解
- 代码示例（带语法高亮）
- 实践提示
- 学习目标

### 第 4 步：标记完成

阅读完课时内容后，点击 **标记完成** 按钮记录学习进度。

## 课程阅读器功能

课程阅读器提供了丰富的阅读辅助功能：

### Markdown 渲染

- 完整的 Markdown 语法支持
- 标题、列表、表格、引用块
- 代码块自动语法高亮（支持 Python、C、C++、C#、SQL 等）
- 链接和图片展示

### 代码高亮

支持多种编程语言的语法高亮显示：
- Python
- C / C++
- C#
- SQL
- JavaScript
- 以及其他常见语言

### 代码复制

每个代码块右上角提供 **复制按钮**，一键复制代码到剪贴板，方便你在练习中使用。

### 目录导航

课程页面提供目录导航（TOC），方便快速跳转到课程的不同章节。

### 前置条件标注

每门课时会标注所需的前置知识，帮助你判断是否具备学习条件。

## 学习进度追踪

### 完成度追踪

系统自动记录你的学习进度：

- **课时级别** -- 每个课时的完成状态（未开始 / 已完成）
- **模块级别** -- 模块完成百分比
- **技术栈级别** -- 整条路线的学习进度

### 仪表盘概览

在首页仪表盘中可以查看：

| 指标 | 说明 |
|------|------|
| 课程完成率 | 已完成课时数 / 总课时数 |
| 连续学习天数 | 连续每天学习的天数 |
| 每日学习概览 | 今日完成的课时和练习 |
| 练习平均分 | 所有练习的平均评测分数 |

### 数据存储

学习进度存储在本地 SQLite 数据库中：
- 路径：`%APPDATA%/DevLearnerAI/data/app.db`
- 数据库使用 WAL 模式，确保数据安全
- 旧版数据库会自动迁移

## 课程笔记

### 添加笔记

在课程阅读页面中，你可以为每个课时添加学习笔记：
- 记录重要概念

- 可随时编辑和更新
- 笔记存储在数据库中，不会丢失

## 学习建议

### 1. 坚持每日学习

系统会记录你的连续学习天数。建议每天至少完成一个课时，保持学习连续性。

### 2. 理论结合实践

每学完一个知识点后，前往 **练习中心** 完成对应的练习题，巩固所学知识。

### 3. 利用 AI 导师

遇到不理解的概念时，使用 **AI 导师** 功能：
- 点击"解释当前课程"，让 AI 详细讲解
- 直接提问你困惑的知识点
- 请求更多示例和练习

### 4. 循序渐进

按照推荐的学习顺序，从基础到进阶逐步学习。注意查看每个课时的前置依赖。

### 5. 完成融合项目

学完基础知识后，通过 **融合项目** 将零散知识串联起来，培养工程实践能力。
