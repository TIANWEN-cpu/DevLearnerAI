## 仪表盘概览

学习仪表盘是 DevLearner AI 的首页，提供学习数据的全面概览。它是你每次打开应用时看到的第一个页面，帮助你快速了解学习状态和下一步计划。

### 仪表盘包含的信息

| 区域 | 内容 |
|------|------|
| 进度概览 | 课程完成率、练习完成情况 |
| 学习统计 | 练习平均分、各技术栈进度 |
| 每日概览 | 今日学习内容和成果 |
| 连续天数 | 连续学习天数统计 |
| 快速导航 | 直达各技术栈的入口 |

## 学习进度追踪

### 课程完成率

仪表盘显示你的整体课程完成率，计算方式：

```
完成率 = 已完成课时数 / 总课时数 x 100%
```

- 总课时数：130+ 课时（所有技术栈）
- 已完成课时：你标记为"已完成"的课时数
- 完成率以百分比和进度条形式展示

### 分技术栈进度

仪表盘展示每个技术栈的独立进度：

| 技术栈 | 追踪内容 |
|--------|----------|
| Python | 已完成课时 / 总课时 |
| C 语言 | 已完成课时 / 总课时 |
| C++ | 已完成课时 / 总课时 |
| C# | 已完成课时 / 总课时 |
| 数据库 | 已完成课时 / 总课时 |
| 算法 | 已完成课时 / 总课时 |
| 融合项目 | 已完成项目 / 总项目 |
| 实战项目 | 已完成项目 / 总项目 |

### 练习完成情况

- 已完成练习数 / 总练习数
- 各难度等级的完成情况
- 最近完成的练习列表

## 统计数据解读

### 练习平均分

仪表盘展示你所有练习的平均评测分数（0-100 分制）。

分数含义：
| 分数段 | 水平 | 建议 |
|--------|------|------|
| 90-100 | 优秀 | 挑战更高难度的练习 |
| 70-89 | 良好 | 巩固薄弱知识点 |
| 50-69 | 一般 | 重做相关课程和练习 |
| 0-49 | 需加强 | 建议先学习相关课程 |

### 技能雷达图

部分版本提供技能雷达图，直观展示各技术栈的掌握程度。

## 连续学习天数

### 什么是连续学习天数

连续学习天数（Streak）是你连续每天使用 DevLearner AI 学习的天数。它是一个重要的学习动力指标。

### 计算规则

- 每天至少完成一个学习动作（阅读课时、完成练习、与 AI 对话）即算作当天有学习
- 连续天数从最近一次学习日开始往回计算
- 如果某天没有学习记录，连续天数重置为 0

### 保持连续学习

- 每天花 15-30 分钟学习
- 即使时间有限，至少阅读一个课时或完成一道练习
- 利用仪表盘提醒自己保持学习连续性

## 快速导航

### 技术栈入口

仪表盘提供直达各技术栈的快速导航入口：

- 点击技术栈名称直接进入对应的学习路径
- 显示每个技术栈的当前进度
- 高亮推荐的下一步学习内容

### 快捷操作

| 操作 | 说明 |
|------|------|
| 继续学习 | 跳转到上次未完成的课时 |
| 练习中心 | 进入练习列表 |
| AI 导师 | 打开 AI 对话 |
| 融合项目 | 查看项目列表 |

## 数据导出与备份

### 数据存储位置

所有学习数据存储在本地：
- 数据库路径：`%APPDATA%/DevLearnerAI/data/app.db`
- 导出目录：`%APPDATA%/DevLearnerAI/exports/`

### 手动备份

定期备份数据库文件以防数据丢失：

1. 关闭 DevLearner AI 应用
2. 复制 `%APPDATA%/DevLearnerAI/data/app.db` 到安全位置
3. 备份完成

### 数据恢复

如果需要恢复数据：

1. 关闭 DevLearner AI 应用
2. 将备份的 `app.db` 文件复制回 `%APPDATA%/DevLearnerAI/data/` 目录
3. 重新启动应用

### 数据库迁移

旧版数据库（`db/learner.db`）会自动迁移到新路径，无需手动操作。
