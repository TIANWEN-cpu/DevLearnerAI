# DevLearner AI 6.0 项目说明

## 一句话介绍

**DevLearner AI** 是一款 AI 驱动的桌面编程学习平台，将 8 条技术路线、130 个课时、176 道练习和 AI 实时辅导整合到一个离线可用的桌面应用中。

---

## 为什么做这个项目

学编程最大的痛点不是缺少教程，而是：

- **看完不会写** -- 缺少动手练习和即时反馈
- **卡住没人帮** -- 一个问题卡几小时，学习热情消耗殆尽
- **知识是碎片的** -- 学了 Python 语法，不知道怎么串成一个项目
- **平台割裂** -- 学语法用一个网站，刷题用另一个，问 AI 又开一个窗口

DevLearner AI 的答案：**一个工作台，全搞定。**

---

## 项目定位

面向编程初学者和进阶学习者的桌面应用，不是"数据处理工具 + 学习器"混合体，而是清晰拆成三条主线：

- **Python 主线**：语言、结构、调试、测试、工程意识
- **数据库主线**：建模、SQL、SQLite、ORM
- **融合项目线**：把 Python 与数据库能力合并成真实项目

---

## 核心功能

### AI 智能导师

不是简单的聊天机器人。AI 导师知道你正在学哪门课、做到哪道题，能基于你的学习上下文给出精准指导。

- 支持 OpenAI 兼容 API，可切换不同端点和模型
- 多会话管理：按主题拆分对话
- 快捷提问：解释当前课程 / 分析当前代码 / 拆解当前项目
- AI 工作台 + 侧边助手双模式

### 代码执行沙箱

在安全的沙箱中运行代码，不怕写错搞坏系统。

- Python 实际运行 + AST 预检拦截危险代码
- C / C++ / C# 结构检查评测
- SQL 真实数据库查询验证
- 输出限制 + 超时控制，安全可靠

### 沉浸式课程体系

130 个课时，覆盖从零基础到工程实战的完整知识地图。

- **8 条学习路线**：Python / C / C++ / C# / 数据库 / 算法 / 融合项目 / 实战项目
- Track > Module > Lesson 三级结构
- Markdown 渲染 + 代码语法高亮
- 课程完成度追踪 + 笔记功能

### 交互式练习

176 道精选练习题，覆盖全部 8 条技术路线。

- 多语言支持：Python / C / C# / SQL
- 自动评测：运行结果 + 语法检查 + 结构检查 + 测试用例通过率
- 按技术栈和难度筛选
- 草稿自动保存，中途退出不丢失

### 实战项目集

10 个真实项目，把零散技能组装成完整作品。

- 从 Todo CLI 到 AI 对话助理，由浅入深
- 跨语言融合项目，培养工程思维

### 学习仪表盘

一眼看清学习全貌：进度追踪、完成统计、每日概览、连续天数、练习平均分。

### 算法可视化

交互式动画，看见排序和查找的每一步。

## 技术栈

| 类别 | 技术 |
|:---|:---|
| 语言 | Python 3 |
| GUI 框架 | PyQt5 |
| 数据库 | SQLite（WAL 模式 + 线程锁） |
| Markdown 渲染 | mistune |
| 语法高亮 | Pygments |
| AI 接口 | OpenAI 兼容 API（支持自定义端点） |
| 凭证管理 | keyring / Windows Credential Manager |
| 打包 | PyInstaller |
| 代码检查 | Ruff（linter + formatter） |
| 测试 | pytest + coverage（1000+ 测试用例） |

## 运行方式

```bash
# 安装依赖
pip install -r requirements.txt

# 启动应用
python main.py

# 开发模式（DEBUG 日志）
python dev_main.py

# 打包
python scripts/build/build_dev_exe.py
```
