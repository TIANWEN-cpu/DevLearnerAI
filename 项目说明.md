# DevLearner AI 6.0 项目说明

## 项目定位
这是一个面向编程学习的桌面应用，不再是“数据处理工具 + 学习器”混合体，而是清晰拆成三条主线：

- Python 主线：语言、结构、调试、测试、工程意识
- 数据库主线：建模、SQL、SQLite、ORM
- 融合项目线：把 Python 与数据库能力合并成真实项目

## 当前版本已经完成的重构
- 统一启动入口：`main.py` 与 `dev_main.py` 现在都启动同一套学习器
- 去掉课程层面的 Python / 数据库混杂结构
- 新建 `app/` 分层代码结构
- 新建 `content/metadata/course_map.json` 统一管理课程元数据
- 新建 `content/metadata/exercises.json` 统一管理练习元数据
- 新增首页、学习路径、练习中心、融合项目、算法动画五个核心页面
- 数据库存储改为围绕课程进度、练习记录、笔记和 AI 配置服务
