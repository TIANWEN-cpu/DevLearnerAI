# DevLearner AI 6.0 项目说明

## 项目定位
这是一个面向编程学习的桌面应用，不再是“数据处理工具 + 学习器”混合体，而是清晰拆成三条主线：

- Python 主线：语言、结构、调试、测试、工程意识
- 数据库主线：建模、SQL、SQLite、ORM
- 融合项目线：把 Python 与数据库能力合并成真实项目

## 当前版本已经完成的重构
- 统一启动入口：`main.py` 与 `dev_main.py` 现在都启动同一套学习器
- 去掉课程层面的 Python / 数据库混杂结构
- 新建 `app/` 分层代码结构
- 新建 `content/metadata/course_map.json` 统一管理课程元数据
- 新建 `content/metadata/exercises.json` 统一管理练习元数据
- 新增首页、学习路径、练习中心、融合项目、算法动画五个核心页面
- 数据库存储改为围绕课程进度、练习记录、笔记和 AI 配置服务

## 功能特性

### 学习系统
- **8 条学习路线**：Python、C、C++、C#、数据库、算法、融合项目、实战项目
- **130 课时**：覆盖从基础语法到工程实战的完整知识体系
- **结构化元数据**：每课附带难度、预估学时、前置依赖和学习目标
- **AI 辅导**：集成 Claude AI 智能问答，实时解答学习问题

### 练习系统
- **176 道练习题**：覆盖全部 8 条学习路线
- **多语言支持**：Python、C、C#、SQL 等多语言练习
- **自动判题**：AST 分析 + 测试用例自动验证
- **难度分级**：基础、进阶、综合三级难度标签
- **SQL 实战**：内置 SQLite 测试数据库，支持真实查询验证

### 项目实战
- **10 个真实项目**：从 Todo CLI 到 AI 对话助理，由浅入深
- **融合项目**：跨语言综合项目，培养工程能力

### 界面与体验
- **桌面应用**：基于 PyQt5 的原生桌面体验
- **代码高亮**：支持多种语言的语法高亮
- **课程阅读器**：带目录导航和代码复制功能
- **学习仪表板**：进度追踪、目标设置和成绩图表
