# Screenshots

本目录用于存放 DevLearner AI 的界面截图，供 README.md 和项目说明.md 引用。

## 需要的截图

| 文件名 | 对应界面 | 说明 |
|--------|----------|------|
| `dashboard.png` | 学习仪表盘 | 进度追踪、目标统计、成绩图表 |
| `course-reader.png` | 课程阅读器 | Markdown 渲染、代码高亮、目录导航 |
| `practice.png` | 练习中心 | 题目列表、代码编辑、自动判题 |
| `ai-chat.png` | AI 工作台 | 实时对话、代码解释、学习建议 |
| `learning-path.png` | 学习路径 | 路线选择、进度追踪 |
| `integration.png` | 融合项目 | 跨语言综合项目 |

## 如何添加截图

1. 运行应用：`python main.py`
2. 导航到对应页面
3. 使用截图工具（如 Windows Snipping Tool）截取界面
4. 保存为 PNG 格式，按上表命名
5. 放入此目录
6. 提交 PR

## 截图规范

- 分辨率：至少 1280x800
- 格式：PNG
- 内容：展示功能核心界面，避免包含个人信息
- 命名：使用小写英文 + 连字符，如 `dashboard.png`
